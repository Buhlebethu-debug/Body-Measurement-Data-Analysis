# Body Measurement Data Analysis

## Overview
This project analyses a structured dataset of body measurements to identify patterns, variation and relationships across different variables.

The dataset includes multiple physical measurements collected from over 2,000 individuals.

## Objectives
- Understand variation across different body measurements
- Identify patterns and relationships between variables
- Interpret summary statistics to generate insights

## Data Preparation
- Cleaned the dataset to ensure consistency
- Reviewed variables and measurement distributions
- Worked with structured tabular data

## Analysis
- Analysed averages, minimums, and maximums
- Explored variation across features such as waist, biceps, and thighs
- Compared different measurement types

## Key Insights
- Soft tissue measurements (e.g. waist, biceps) showed higher variation
- Skeletal measurements were more consistent across individuals
- Clear differences exist in body composition across the dataset

## Tools Used
- Data Analysis
- Exploratory Data Analysis (EDA)

## Skills Demonstrated
- Working with structured datasets
- Data interpretation
- Pattern recognition
- Turning data into insights
